### Instruções

1. Criar o HTML/CSS responsivo com base no layout navegável. 

2. Nós queremos avaliar o seu nível de conhecimento em design responsivo (Responsive Web Design) usando HTML5, CSS3 e grids. Caso queira, utilize pré-processadores.

3. O grid do layout é baseado no Grid System do Bootstrap, mas não copie todo o bootstrap  para o projeto.

### Atenção

* Os recortes das imagens estão na pasta "slices", mas você é livre para usar um http://lorempixel.com no lugar.
* Hospede o CSS dentro da pasta "css".
* Não é necessário fazer o carrossel/rotate funcionar.
* Não é necessário que a fonte seja exatamente a mesma usada na arte. Se quiser mantê-la, é a "Helvetica". ;)
* É permitido usar qualquer editor de código e consultar a internet.
* Suporte de Browsers: IE9+, Chrome, Safari, Firefox.
